# MyTube Downloader

تطبيق Android لتحميل فيديوهات YouTube بجودة عالية.

## المميزات
- تحميل فيديوهات YouTube بجودة عالية
- دعم مشاركة الروابط من التطبيقات الأخرى
- واجهة بسيطة وسهلة الاستخدام
- تحميل مباشر إلى مجلد التنزيلات

## كيفية الاستخدام
1. انسخ رابط فيديو YouTube
2. افتح التطبيق وأدخل الرابط
3. اضغط على زر التحميل
4. انتظر حتى يكتمل التحميل

## الملاحظات
- يتطلب صلاحيات التخزين لحفظ الفيديوهات
- التطبيق متوافق مع Android 7.0 فأعلى
